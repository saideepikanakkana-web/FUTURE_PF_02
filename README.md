# 🎥 AI UGC Ad Content Generator  
### Prompt Engineering Task 2 – Future Interns (2026)

---

## 📌 Project Overview  
This project demonstrates how AI and Prompt Engineering can be used to create high-converting UGC (User Generated Content) ad scripts.

Unlike traditional ads, UGC ads feel:
- Real  
- Relatable  
- Trustworthy  

This system generates authentic, short-form ad content similar to what top D2C brands and marketing agencies use for Instagram Reels, Shorts, and paid ads.

---

## 🏷️ Product / Business Selected  
Glow Skin Serum (D2C Skincare Product)  

Target Audience: Women aged 18–35  
Platform Focus: Instagram Reels & Shorts  

---

## 🎯 Objectives  
- Build a reusable prompt system for UGC ad creation  
- Generate multiple hooks for better engagement  
- Create short-form ad scripts using proven frameworks  
- Adapt content for different platforms  
- Produce ad-ready content for real marketing use  

---

## 🧠 Problem Statement  
Most ads fail because:
- They look too polished and salesy  
- They don’t feel authentic  
- They fail to connect with real users  

UGC ads solve this by:
- Using real-life storytelling  
- Showing relatable problems  
- Building trust through experience  

---

## ⚙️ Prompt System  

This project uses a structured AI prompt that generates:

- Scroll-stopping hooks  
- UGC-style ad scripts  
- Platform-optimized content  
- Call-to-actions (CTAs)  
- Social media captions  

The prompt ensures consistency, scalability, and high conversion potential.

---

## ✍️ Content Generated  

### 🔹 Hooks  
- Multiple attention-grabbing opening lines  
- Designed to stop scrolling  
- Based on curiosity and pain points  

---

### 🔹 UGC Ad Scripts  
Each script follows a proven structure:

👉 Hook → Problem → Solution → Experience → CTA  

- Written in first-person style  
- Natural and conversational tone  
- Short-form (20–40 seconds)  
- Focused on real user experience  

---

### 🔹 Call-To-Actions (CTAs)  
Different types of CTAs were generated:

- Soft CTA  
- Urgency CTA  
- Offer-based CTA  
- Trust-based CTA  

---

### 🔹 Captions  
- Engaging and short  
- Includes emojis (minimal)  
- Uses relevant hashtags  
- Matches platform tone
